O if do destino

Missão:

Ao seguir sua jornada, você encontra uma encruzilhada. Um velho cavaleiro de armadura enferrujada te avisa:

Só quem souber usar a lógica condicional pode atravessar...

Tarefa:

Crie um formulário com:

Um campo para o nome

Um campo para a idade

Um botão que chama a função verificarDestino()

Essa função deve:

Dizer se a pessoa pode ou não entrar na Guilda dos Devs

Se a idade for maior ou igual a 18, mostrar:

"Seja bem-vindo(a), [nome]. A Guilda dos Devs te recebe de braços abertos!"

Se for menor:

"Sinto muito, [nome], você ainda não tem idade suficiente para entrar na Guilda."
