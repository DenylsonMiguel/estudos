O Labirinto dos Laços

Missão:

Você chega ao Labirinto dos Laços, uma antiga masmorra onde os loops são a única saída. Um escriba ancião deixa uma inscrição na pedra:

Somente aquele que souber repetir com lógica encontrará a saída.

Tarefa:

Crie um botão: Iniciar missão

Ao clicar, o sistema deve exibir:

Passo 1

Passo 2

Passo 3

...

Passo 10

Os passos devem aparecer dentro de uma <div> (tipo um pergaminho mágico!)
